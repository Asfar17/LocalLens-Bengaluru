# Background Image Assets

## bangalore-map.svg

This is a placeholder SVG background representing a stylized city map. 

### To use an actual Bangalore map image:

1. Replace `bangalore-map.svg` with `bangalore-map.jpg` (or `.png`, `.webp`)
2. Recommended specifications:
   - Resolution: 1920x1080 or higher
   - Format: WebP (preferred) or JPEG (compressed)
   - File size: Under 500KB for optimal loading
   - Style: Muted colors work best with glassmorphism overlay

3. Update the reference in `App.tsx` if using a different file extension

### Image optimization tips:

- Use tools like TinyPNG, Squoosh, or ImageOptim
- Consider using WebP format for better compression
- Ensure the image works well with the glass overlay effect
